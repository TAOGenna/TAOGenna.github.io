A - A Symetrical Pizza

The way I found the solution for this problem was through playing with the input. It's no hard tho to arrive to the following relation: \( Answer=\frac{mcm(R,360)}{R}\) or \(Answer=\frac{360}{gcd(R,360)} \).

Maybe the hard thing to realise is that you have to make a parsing on the input in order to get the correct solution. 

def func(A):
	    result = 0
	    n = len(A)
	    while n > 1:
		n = n/2
		result = result + min(A[1,...,n])
	    return result
