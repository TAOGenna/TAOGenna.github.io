# Kenyi Takagui-Perez — Portfolio

Built with [Hugo](https://gohugo.io/). Style inspired by [karinanguyen.com](https://karinanguyen.com/).

## Local development

```bash
hugo server -D
```

Open http://localhost:1313/

## Build for production

```bash
hugo
```

Output goes to `docs/` (configured for GitHub Pages).

## Writing a new blog post

Create a markdown file in `content/writing/`:

```bash
content/writing/my-new-post.md
```

With this frontmatter:

```markdown
---
title: "Your Post Title"
date: 2026-04-04
categories: ["math"]
---

Your content here. Regular markdown works: **bold**, *italic*, 
[links](https://...), code blocks, images.

Inline math: $E = mc^2$

Display math:

$$
\int_0^\infty e^{-x^2} dx = \frac{\sqrt{\pi}}{2}
$$
```

The post auto-appears in the Writing list sorted by date. To include images,
place them next to your `.md` file inside a [page bundle](https://gohugo.io/content-management/page-bundles/):

```
content/writing/my-new-post/
├── index.md
├── diagram.png
└── photo.jpg
```

Then reference them in markdown as `![alt](diagram.png)`.

## Adding a project

Create a markdown file in `content/projects/`:

```markdown
---
title: "Project Name"
image: "my-image.jpg"
description: "Short description shown under the card"
year: "2025"
type: "ai"
order: 1
external: "https://github.com/..."
---
```

Place the project image in `static/Files/`. The `order` field controls display order (lower = first).

## Distill courses

The `courses/` directory (inside `static/`) is auto-generated by the [Distill](https://github.com/TAOGenna/Distill) project. Don't edit it by hand.

To update after generating new courses:
```bash
cd ~/Desktop/kenyi/projects/Distill
uv run distill publish
```

Note: after migrating to Hugo, the Distill publish destination should point to `static/courses/` in this repo.
